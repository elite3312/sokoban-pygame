# sokoban-pygame 0.0.2
an unusual sokoban made with pygame 2.0

# Requirements
python 3.7+  
pygame 2.0+

# How To Play
按空白鍵 => 攻擊 (space key => attack)  
按ESC => 暫停 (ESC => pause)
